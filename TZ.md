# Техническое задание: AI Stain Fix (Пятновыводитель)

> **Составлено по «Ветхому Завету» v2.0**

## 1. НАЗВАНИЕ: AI Stain Fix (Пятновыводитель)

## 2. ОПИСАНИЕ (1 строка)
Фото пятна → AI определяет тип загрязнения → чем вывести + пошаговая инструкция в зависимости от типа ткани.

## 3. PACKAGE ID: com.zubcoder.ai_stain_fix

## 4. EMAIL: ai.stain.fix.app@gmail.com

## 5. ДОМЕН: aistainfix.ru (для privacy/terms)

## 6. ЦВЕТА:
- Primary: #7C4DFF (фиолетовый — ассоциация с чистотой и химией)
- Accent: #00E5FF (голубой — ассоциация с водой и свежестью)

## 7. ЛОГОТИП
Тематика: капля воды с AI-элементом (нейросеть) внутри + эффект «чистоты» (блеск/sparkle). CustomPaint вектор.

## 8. ОНБОРДИНГ:
- Слайд 1: **Сфотографируйте пятно** — Наведите камеру на загрязнение для мгновенного анализа
- Слайд 2: **AI определит тип** — Искусственный интеллект распознает вид пятна и ткань
- Слайд 3: **Получите инструкцию** — Пошаговый рецепт удаления с доступными средствами
- Слайд 4: **Умный помощник** — AI учитывает тип ткани, возраст пятна и доступные средства

## 9. МОНЕТИЗАЦИЯ:
- Бесплатно: 3 анализа в день
- Пакет 1: Капля (10 анализов) — 79 ₽
- Пакет 2: Стирка (25 анализов) — 179 ₽
- Пакет 3: Химчистка (50 анализов) — 329 ₽
- PRO: 149 ₽/мес / 1190 ₽/год

## 10. AI ФУНКЦИИ:
- **Анализ пятна по фото**: определение типа загрязнения (жир, вино, кровь, чернила, ржавчина, трава, кофе и т.д.)
- **Определение ткани**: распознавание типа материала (хлопок, шёлк, синтетика, шерсть, кожа и т.д.)
- **Генерация инструкции**: пошаговый рецепт удаления с учётом типа пятна + ткани
- **Рекомендации средств**: доступные бытовые и профессиональные средства

## 11. BACKEND ENDPOINTS:
- POST /api/v1/ai/analyze — анализ фото пятна (base64 image → тип пятна + ткань + инструкция)
- GET  /health — проверка состояния сервиса

## 12. ЭКРАНЫ:
- **Камера** → фото пятна + выбор типа ткани
- **Результат** → тип пятна, рецепт удаления, список средств, пошаговая инструкция
- **История** → список предыдущих анализов
- **Подписка** → пакеты и PRO
- **Настройки** → язык, тема, поддержка, о приложении

## 13. ДОПОЛНИТЕЛЬНО:
- Реклама: баннер внизу для бесплатного плана
- Допы: нет
- Особенности:
  - НУЛЕВАЯ конкуренция в Google Play
  - Понятный интерфейс для любого человека на планете
  - Огромный вирусный потенциал (пятна — повседневная проблема)
  - Целевая аудитория: домохозяйки, студенты, родители, все кто стирает
